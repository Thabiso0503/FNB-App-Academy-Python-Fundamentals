# 📖 About

This repository documents my learning journey through the **FNB App Academy – Python Fundamentals** course. It contains every practical task and challenge project completed throughout the six lessons, demonstrating the core concepts of Python programming through hands-on application development.

---

# 🎯 Course Overview

Throughout this course I learned to:

- Write Python programs using clean coding practices
- Work with variables and data types
- Manipulate strings using built-in methods
- Perform arithmetic operations and type casting
- Build decision-making programs using conditional statements
- Store and manage data using lists and dictionaries
- Create reusable functions
- Use loops to automate repetitive tasks
- Develop interactive command-line applications
- Apply problem-solving techniques to real-world programming challenges

# 📚 Lessons

| Lesson | Key Concepts | Practical Task | Challenge |
|:------:|--------------|----------------|-----------|
| **1** | Variables, Data Types, User Input, `print()`, `input()`, f-Strings | Student Info Formatter | Concert Ticket Booker |
| **2** | String Manipulation, String Methods, Indexing, Slicing | Username & Message Formatter | Secure Password Hint Tool |
| **3** | Arithmetic Operators, Type Casting, Mathematical Functions | Multi-Function Calculator | South African Fuel Cost Calculator |
| **4** | Conditional Statements, Comparison & Logical Operators | Grade Classifier | Smart ATM Withdrawal Simulator |
| **5** | Lists, Dictionaries, Functions | Contact Book | Contact Book Extension |
| **6** | Loops (`for`, `while`, `range`, `break`, `continue`) | Loop Exercises | Loop Challenge |
