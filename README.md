# 📖 About

This repository documents my learning journey through the **FNB App Academy Python Fundamentals** course. It contains every practical task and challenge project completed throughout Lessons 1–6, showcasing the Python concepts learned and the applications built during the course.

---

# 🎯 Course Overview

Throughout this course I learned how to:

- Work with variables and Python data types
- Manipulate strings using built-in methods
- Perform arithmetic calculations and type casting
- Build decision-making programs using conditional statements
- Store and manage data using lists and dictionaries
- Write reusable functions
- Use loops to automate repetitive tasks
- Develop interactive command-line Python applications

| Lesson | Concepts Learned | Practical Task | Challenge |
